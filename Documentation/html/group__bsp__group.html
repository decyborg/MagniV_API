Detailed Description

BSP (Board Support Package) is the lowest layer of the LIN Stack.

BSP is comprised of codes, which implements the tasks dedicated to specific MCU platform: interrupt service routines, i/o port setup, memory handling and so on. There are four interfaces implemented within the stack package: SCI, SLIC, GPIO and SCI + XGATE, UART.
