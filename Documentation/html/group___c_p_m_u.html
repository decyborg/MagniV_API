unsigned long  FPLL 
)

Searches through possible values to find the best match.

Searches for the values of POSTDIV and SYNDIV given a certain frequency reference, this function only writes the following values on the clock_dividers structure: postdiv, syndiv, vcofrq and error.

Parameters
[in]FrefReference frequency used to find the values
[in]FPLLDesired PLL frequency (bus frequency * 2) in hertz
Returns
Structure with the found values and the error of the setup
